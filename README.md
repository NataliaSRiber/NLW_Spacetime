## Rodando a aplicação localmente
### Requisitos

Para iniciar é necessário ter instalado na máquina as seguintes ferramentas:
[Git](https://git-scm.com), [Node.js](https://nodejs.org/en/) e um editor de código como o [VSCode](https://code.visualstudio.com/).

### Rodando o servidor Back-End

```bash
# Clone este repositório com a chave SSH ou HTTP a depender de como seu git está configurado.

# Acesse a pasta do projeto no terminal
$ cd nlw_spacetime

# Entre na pasta back-end
$ cd server

# Instale as dependências com o comando
$ npm install

## Crie um arquivo .env
	
# Execute a aplicação
$ npm run dev

# O servidor iniciará na porta:3001 - <http://localhost:3001>
```

### Rodando o Front-End 

```bash
# Clone este repositório com a chave SSH ou HTTP a depender de como seu git está configurado.

# Acesse a pasta do projeto no terminal/cmd
$ cd nlw_spacetime

# Entre na pasta front-end
$ cd web

# Instale as dependências
$ npm install

# Execute a aplicação em modo de desenvolvimento
$ npm run dev

# O servidor inciará na porta:3000 - <http://localhost:3000>
```
